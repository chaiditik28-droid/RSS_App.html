🔥 ตั้งค่า Firebase Realtime Database
เก็บข้อมูลบน Cloud — ทุกเครื่องเห็นพร้อมกัน รูปภาพไม่หาย
📋 ขั้นตอนตั้งค่า Firebase (ทำครั้งเดียว ~10 นาที)
ขั้นที่ 1 — สร้างโปรเจกต์
1. ไปที่ console.firebase.google.com
2. กด "Add project"
3. ตั้งชื่อ เช่น RSS-AAM
4. ปิด Google Analytics → กด "Create project"
ขั้นที่ 2 — เปิด Realtime Database
1. เมนูซ้าย → "Build" → "Realtime Database"
2. กด "Create Database"
3. เลือก Region: asia-southeast1 (Singapore)
4. เลือก "Start in test mode" → กด Enable
ขั้นที่ 3 — คัดลอก Database URL
หลัง Enable จะเห็น URL รูปแบบ:
https://rss-aam-default-rtdb.asia-southeast1.firebasedatabase.app กด Copy URL นี้ไว้
ขั้นที่ 4 — แชร์ให้ทีม
1. วาง URL ในช่องด้านล่าง → กด เชื่อมต่อ
2. อัปโหลดไฟล์ HTML ขึ้น Google Drive
3. ส่ง link ให้ทุกคน → ใส่ Firebase URL เดียวกัน
4. ข้อมูล sync ทุก 8 วินาที ✅
⚠️ หมายเหตุ Test Mode: Database จะ public 30 วัน ถ้าต้องการป้องกันให้ไปที่ Rules แล้วเปลี่ยนวันหมดอายุ หรือใส่ rule ตรวจสอบ authentication ครับ
